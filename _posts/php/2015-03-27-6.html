5.gem改为淘宝源

安装好Ruby以后gem的源一般默认为http://rubygems.org/，这个源有时候不太好用，我们可以改成国内淘宝的。目前我在windows下开发，打开cmd窗口，敲以下命令即可。

                显示当前使用的sources
                gem sources
                删除默认sources
                gem sources -r http://rubygems.org/
                添加淘宝sources
                gem sources -a http://ruby.taobao.org/
                更新缓存
                gem sources -u

6.Add HTTPS support for Youku

All Youku videos went blank on HTTPS forums, because "player.youku.com" does not have SSL support. But actually Youku does support HTTPS, with a different URL.

                e.g.
                Original link: http://v.youku.com/v_show/id_XMTU0MzMyNjg2NA.html
                HTTP player: http://player.youku.com/embed/XMTU0MzMyNjg2NA
                HTTPS player: https://players.youku.com/player.php/sid/XMTU0MzMyNjg2NA/v.swf

https://github.com/s9e/flarum-ext-mediaembed/issues/5

7.通过url参数控制youtube视频自动播放，循环播放，无控制按键

参考：https://developers.google.com/youtube/player_parameters#loop

                关键参数：
                autoplay=1&loop=1
                rel=0&loop=1&autoplay=1&controls=0&showinfo=0&autohide=1
                例子：
                https://www.youtube.com/embed/XS7jXQAnG5k?rel=0&?autohide=1&iv_load_policy=3&enablejsapi=1&loop=1&autoplay=1
